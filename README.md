# 복기노트 iOS PWA (빨초 아이콘 적용본)

포함 파일:
- index.html
- manifest.webmanifest
- service-worker.js
- apple-touch-icon.png
- icon-192.png
- icon-512.png

사용 방법:
1. GitHub 저장소 `bokgi-note`로 이동
2. 기존 파일을 지우지 말고, 이 폴더 안 파일들을 그대로 다시 업로드
3. 같은 이름 파일은 덮어쓰기
4. Commit changes
5. 1-5분 기다린 뒤 Safari에서 사이트 열기
6. 기존 홈화면 아이콘 삭제
7. 다시 `홈 화면에 추가`

아이콘이 안 바뀌면:
- 사이트 주소 뒤에 `?v=3` 또는 `?v=4`를 붙여서 다시 접속 후 홈화면 추가
