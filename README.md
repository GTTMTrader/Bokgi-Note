# 복기노트 iOS PWA - Claude 수정본 적용

Claude에서 수정한 HTML을 `index.html` 이름으로 정리하고, 빨초빨초 아이콘/PWA 파일을 같은 이름으로 다시 묶은 버전입니다.

GitHub에 올릴 파일:
- index.html
- manifest.webmanifest
- service-worker.js
- apple-touch-icon.png
- icon-192.png
- icon-512.png
- README.md

업로드:
1. ZIP 압축 풀기
2. GitHub `bokgi-note` 저장소
3. Add file → Upload files
4. 압축 푼 내부 파일 전부 업로드
5. 같은 이름 파일은 덮어쓰기
6. Commit changes
7. 1-5분 대기
8. Safari에서 `https://gttmtrader.github.io/bokgi-note/?v=5` 접속
9. 기존 홈화면 아이콘 삭제 후 다시 홈 화면에 추가
